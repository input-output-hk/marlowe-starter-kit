Example of Using Marlowe Runtime with a CIP30 Wallet

This example shows how to use a Babbage-compatible CIP-30 wallet such as Eternl to sign Marlowe transactions. The example contract here is a zero-coupon bond, but both the lender and borrower use the same wallet, for simplicity of the example. Use the preprod testnet with this web application, which was tested against Eternl 1.11.13.
